# .github

Community Health Files for Github.

This is the overlay repo for my personal repositories on Github.

## What is this?

The .github repository is a special repository on GitHub that allows you to
centralize and manage default community health files and configurations
for all repositories within your personal account or organization.

You can define standard templates and settings that will automatically
apply to any repository lacking its own specific configurations.

See Github Docs: [Creating a default community health file](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).
